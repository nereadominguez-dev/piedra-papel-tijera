# Piedra · Papel · Tijera (Java + Web)

Proyecto realizado durante mis estudios de **Desarrollo de Aplicaciones Web (DAW)**.

//Jugar online
- Versión web (GitHub Pages): https://nereadominguez-dev.github.io/piedra-papel-tijera/

// Código fuente
- Repositorio: https://github.com/nereadominguez-dev/piedra-papel-tijera

// Descripción
Este proyecto incluye dos versiones del juego clásico **Piedra, Papel o Tijera**:
- **Java (consola):** versión original desarrollada en BlueJ.
- **Web (HTML/CSS/JS):** versión adaptada para jugar desde el navegador con interfaz visual, marcador y estados de victoria/derrota/empate.

//Estructura del proyecto
- `java/` → versión Java por consola
- `docs/` → versión web publicada en GitHub Pages (HTML, CSS, JavaScript)

// Qué he practicado
- Lógica de programación (condicionales, comparaciones y aleatoriedad)
- Control de flujo y validación de entradas
- Maquetación web y diseño responsive
- Manipulación del DOM con JavaScript
- Control de versiones con Git/GitHub
- Publicación del proyecto con GitHub Pages

// Cómo ejecutar -> Java (consola)
1. Abrir el archivo dentro de `java/`
2. Compilar y ejecutar:
   - `javac PiedraPapelTijera.java`
   - `java PiedraPapelTijera`

// Web (local)
1. Abrir `docs/index.html` en el navegador
2. (Opcional) usar Live Server en VS Code para una mejor experiencia
